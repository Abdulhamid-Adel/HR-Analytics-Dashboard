# HR Analytics Dashboard
## Dashboard Preview
## Overview
This project is an interactive HR Analytics Dashboard developed using Power BI to analyze employee data and support data-driven decision-making.

## Tools
- Power BI
- Power Query
- DAX
- Data Modeling

## Features
- Employee Overview
- Department Analysis
- Salary Analysis
- Gender Distribution
- KPI Cards
- Interactive Filters

## Key Insights
- Total Employees: 2,500
- Male Employees: 1,211
- Female Employees: 1,289
- Marketing has the highest total net salary.
- Finance has the lowest total net salary.
